
# NIXOS
```
gcl -q https://github.com/kln-os/dotfiles.git ~/.dotfiles 
mkdir -p ~/.config/run-or-raise
ln -s ~/.dotfiles/run-or-raise/shorcuts.conf ~/.config/run-or-raise/shortcuts.conf
ln -s ~/.dotfiles/ssh/config ~/.ssh/config
```
