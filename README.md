
# NIXOS
```
gcl https://github.com/kln-os/dotfiles.git ~/dotfiles 
ln -s ~/dotfiles/run-or-raise ~/.config/run-or-raise
ln -s ~/dotfiles/ssh/config ~/.ssh/config
```
# MACOS

